# Hey around

Это мобильное приложение, которое представляет из себя события на карте.          
Любой человек может как создать какое-то событие, так и присоединитсья к уже существующему. Также есть личный кабинет, в котором находятся события, в которых участвует пользователь. Имеется поиск по событиям.

## Какие проблемы решает приложение?
* Отсутствие планов и занятий на день
* Постоянное нахождение дома
* Отсутствие коммуникации
* Одиночество
